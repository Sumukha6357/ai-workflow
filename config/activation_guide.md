# 🚀 AI Hub Setup: Beginner's Guide 🚀

Follow these 3 easy steps to connect this AI Hub to any New Machine.

## Step 1: Clone the Hub
*   Download or `git clone` this `ai-governance` folder to `C:\Proj\ai-governance`. 
*   *Note: Using this exact path ensures everything works instantly.*

## Step 2: Paste the AI Rules
*   Open VS Code and click the **Antigravity (Gemini)** icon in the sidebar.
*   Go to **Settings** (Gear Icon) -> **Rules**.
*   Open the file [global_rules.md](global_rules.md) and copy its contents into the AI Rules box.

## Step 3: Run the Link Hub Script
*   Open any project you want to work on.
*   Run this command in your VS Code terminal (PowerShell):
    `powershell -ExecutionPolicy Bypass -File 'C:\Proj\ai-governance\config\link_hub.ps1'`
*   **Result**: This will "Neural Link" your AI's brain to this Hub and sync all your 19+ skills.

---
### 🎉 That's it!
Now, you can just start the project with the `/init-project` command.
I will read your Hub and follow your rules automatically!
